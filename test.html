Hotel Management System

This page is a quick summary version of the documentation and can be used for testing content blocks, layout previews, or lightweight client presentations.

What the system currently includes

  • Authentication with login using username or email
  • Basic dashboard
  • User and role management with dynamic permissions
  • General settings for app name, currency, timezone, and business preferences
  • Language management with RTL and LTR support
  • Room master data including bed types, booking types, room types, facilities, and room sizes
  • Room management for buildings and floors
  • Customer management for basic guest records

Planned improvements

  • Booking and reservation management
  • Room status tracking
  • Simple payment tracking

Technology stack

  • PHP 8.2+
  • Laravel 12
  • Blade Templates
  • MySQL / MariaDB
  • Vite
  • Tailwind CSS 4
  • JavaScript / jQuery
